# Event Scrapper

### Run Server

1) clone repo
```git clone https://github.com/thegreatsantini/event-scrapper.git```

2) install dependencies
```npm i````

3) run server
```nodemon```

### Run Front end

1) open a new tab move to client directory
```cd client```

2) install dependencies
```npm i```

3) start front end
```npm start```
